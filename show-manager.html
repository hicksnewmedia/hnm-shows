How to Update Your Show Page

The Show Manager saves your changes in this browser. When you're ready, download the updated file, rename it to index.html, and drop it into the correct show folder in your repo — then push to deploy live in ~30 seconds.

1
Make all your changes in the Videos, Analytics, and Hosts sections
2
Click the Download button below for the show you want to update
3
Rename the downloaded file to index.html, then replace the file in the matching show folder (e.g., team-no-sleep/index.html)
4
Run git add . && git commit -m "update" && git push — live in ~30 seconds via GitHub Actions
Auto-Save
Your changes are automatically saved to this browser as you work. They'll be here next time you open the Show Manager — no login needed.
How To Use This Manager
A quick guide to managing your three show pages.
Adding Videos
  1. Go to youtube.com/@jameshicks and open the episode you want to add
  2. Copy the video ID from the URL bar — it's the text after ?v=
    youtube.com/watch?v=THIS_PART
  3. Paste the ID into the YouTube Video ID box
  4. Add a title and date, then click Add Episode
  5. Check Mark as Featured for up to 2 episodes you want highlighted
  6. Go to Export → Download to save the updated page
Updating Analytics
  1. Open studio.youtube.com and go to the Analytics section
  2. Copy your Subscribers, Views, and Watch time totals
  3. Paste them into the Analytics section here — you can use formats like 14.2K or 1.8M
  4. Update the age and location percentages from your Audience tab in YouTube Studio
  5. Export the updated page when done
Updating Host Bios & Social Links
  1. Click Co-Host Bios in the left sidebar
  2. Update each host's bio and paste in their social media profile URLs
  3. Leave a social link field empty to hide that icon
  4. Export when done
Publishing Your Pages

After downloading an updated HTML file, you can host it anywhere — Netlify Drop (free, drag-and-drop), GitHub Pages, or your existing web host. Each file is fully self-contained with no dependencies — just upload the single HTML file and it works.
